LSTM Gate Mechanism

🔴

Forget Gate

Decides which historical price information to discard. Uses sigmoid activation σ(Wf·[ht-1, xt] + bf)

ft = σ(Wf·[ht-1, xt] + bf)
🟢

Input Gate

Determines which new market data to store in the cell state — captures momentum and volatility shifts.

it = σ(Wi·[ht-1, xt] + bi)
🔵

Output Gate

Filters the cell state to produce the final predicted price for the next trading interval.

ot = σ(Wo·[ht-1, xt] + bo)
